# Weather App (Open-Meteo)

## Opis
Aplikacja pogodowa stworzona we Flutterze wykorzystująca API Open-Meteo.

## Funkcje
- Aktualna pogoda
- 7-dniowa prognoza
- Ulubione miasta
- Tryb offline (Hive)
- Wykres temperatur
- Firebase Analytics + Crashlytics

## Technologie
- Flutter
- Open-Meteo API
- Hive
- Provider
- Firebase

## Instalacja
flutter pub get  
flutter run
